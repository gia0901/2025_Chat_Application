3. fd_set và connect() ?
An fd_set is a data structure used in conjunction with the select() or pselect() system calls
to monitor multiple file descriptors (including sockets) for readiness for I/O operations. 
It allows a program to efficiently manage multiple concurrent connections or I/O streams 
without needing to create a separate thread or process for each one.
